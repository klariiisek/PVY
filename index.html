2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Keep code modular Break your code into small, reusable functions or modules. This makes it easier to maintain, test, and debug.
  2. Use descriptive names Use meaningful names for variables, functions, and classes. This makes your code easier to read and understand.
  3. Comment your code Add comments to explain complex code or to provide context for future developers. This makes your code more maintainable.
  4. Use consistent formatting Follow a consistent coding style, such as indentation, spacing, and naming conventions. This makes your code more readable and maintainable.
  5. Test your code Write unit tests to verify that your code works as expected. This helps catch bugs early and ensures that your code is reliable.

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script
<script>console.log('Hello');</script>
// 2. Odkázat na externí soubor JavaScriptu
<script src="scripts.js"></script>
// 3. Umístit kód přímo do atributu onclick HTML elementu
<button onclick="alert('Hello')">Click me</button>
// 4. Přidat kód do souboru s událostmi (event listener)
document.getElementById('button').addEventListener('click', function() {
  alert('Hello');
});

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty (0)
let x; // bez přiřazení hodnoty (undefined)
let x = null; // explicitní přiřazení prázdné hodnoty (null)
let x = 5; // celé číslo (typ number)
let y = 3.14; // reálné číslo (typ number)

let text = "Hello, world!"; // typ string
let prestupny = false; // přiřazení booleanské hodnoty (false)

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

= (přiřazení): let x = 5
+= (přičítání): x += 3 // x = x + 3
-= (odečítání): x -= 2 // x = x - 2
*= (násobení): x *= 4 // x = x * 4
/= (dělení): x /= 2 // x = x / 2

Logické

&& (a): true && false // false
|| (nebo): true || false // true
! (negace): !true // false

Porovnávací

== (rovná se): 5 == "5" // true (porovnává hodnoty, ignoruje typ)
=== (striktní rovnost): 5 === "5" // false (porovnává i typy)
!= (nerovná se): 5 != 4 // true
!== (striktní nerovnost): 5 !== "5" // true
> (větší než): 5 > 3 // true
< (menší než): 5 < 3 // false
>= (větší nebo rovno): 5 >= 5 // true
<= (menší nebo rovno): 3 <= 5 // true

Bitové

& (bitový AND): 5 & 1 // 1
| (bitový OR): 5 | 1 // 5
^ (bitový XOR): 5 ^ 1 // 4
~ (bitová negace): ~5 // -6
<< (bitový posun doleva): 5 << 1 // 10
>> (bitový posun doprava): 5 >> 1 // 2

Typové

typeof (typ proměnné), instanceof (kontrola instance)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/*Na webové stránce se zobrazí popup okno s varováním, které informuje uživatele, že nezadal správné heslo. Uživatel musí kliknout na tlačítko „OK“, aby okno zavřel a pokračoval. */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt('Zadej své heslo:');

confirm()

/* Na webové stránce se zobrazí popup okno s dotazem, zda chce uživatel přejít na stránky Seznam.cz. Pokud uživatel klikne na „OK“, prohlížeč přesměruje uživatele na zadanou URL. Pokud klikne na „Zrušit“, zůstane na aktuální stránce. */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:

  1. Rušivost pro uživatele - popup okna mohou být rušivá a omezovat uživatelskou zkušenost.
  2. Zabezpečení a důvěra - časté používání popup oken může vzbuzovat podezření a nedůvěru u uživatelů, zejména pokud se používají k získání citlivých informací, jako jsou hesla nebo osobní údaje
